# Rick and Morty Character Explorer

A responsive web application built with Next.js and TypeScript that allows users to browse, filter and explore characters from the Rick and Morty universe using the public Rick and Morty API.

## Features

* Character listing with images and key information
* Filter by status (Alive, Dead, Unknown)
* Filter by gender
* Search characters by name
* Pagination system
* Responsive design for desktop and mobile devices

## Technologies

* Next.js
* React
* TypeScript
* REST API
* CSS

## Installation

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Project Purpose

This project was developed as part of a Web Interface Programming course and demonstrates the development of a complete React/Next.js application consuming external APIs, implementing filters, pagination and dynamic user interfaces.

## API

Data provided by the Rick and Morty API:

https://rickandmortyapi.com/
